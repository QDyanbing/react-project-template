# React Project Template

一个用于快速创建 React 应用的工程模板，提供统一的开发环境、工程配置、项目结构和常用基础能力。

## 基础环境

- Node.js 版本不低于 24，Volta 固定使用 `24.18.0`。
- 默认使用 Utoo（`ut`）管理依赖，pnpm 作为回退方案。
- 依赖版本根据兼容策略使用 `^` 或 `~`，安装时解析范围内的最新版本，不提交锁文件。
